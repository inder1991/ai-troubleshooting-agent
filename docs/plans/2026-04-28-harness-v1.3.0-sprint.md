**End-to-end description:** Today every regex-only check in the
substrate has the same shape: `re.compile(r'literal_module\.literal_fn')`
matching against raw source text. That misses three large families of
real-world code: aliased imports (`import x as y`), AST-shape variants
(`Annotated[T, Depends(f)]` vs `Depends(f)` defaults), and multi-line
calls. The audit found 30 places this matters. Rather than fix each in
isolation, v1.3.0 introduces three shared building blocks (an import
tracker, an AST-walk helper for "any subprocess-shaped call", a
canonical "is this a logger call?" predicate) and rewrites the affected
checks to consume them. v1.3.1 cleans up the long tail that the shared
building blocks don't reach.

**Tech stack:** Python 3.11+, `ast`, `tomllib`, pytest, ruff.

### Story S1 — `_common.ImportTracker` helper

**Audit IDs closed:** S-A4 (eval/exec aliased), S-AS3 (`from httpx import Client`), S-AS4 (`from time import sleep`), S-AS6 (`import requests as r`), S-DB5 (`from sqlalchemy import text`).

**Why:** Today every check that wants to detect a "banned module" or "banned function" matches `module.function(...)` literal text. Aliased imports (`import x as y; y(...)`) and from-imports (`from m import f; f(...)`) slip every such check. Centralizing the bound-name resolution in a single helper that every check consumes eliminates the gap once.

**Files:**

- Create / modify: `.harness/checks/_common.py` (add class).
- Create: `tests/harness/test_import_tracker.py`.

**Step 2: Implement**

```python
class ImportTracker:
    """Map bound names in a module's scope back to their canonical
    fully-qualified module path.

    Resolves:
      * `import X`            → name X → module "X"
      * `import X as Y`       → name Y → module "X"
      * `from X import Y`     → name Y → module "X.Y"
      * `from X import Y as Z`→ name Z → module "X.Y"

    .canonical_for(func_expr) walks an AST node (Name or Attribute
    chain) and returns the dotted canonical path. Used by callers to
    detect "this expression resolves to module X.Y" without caring
    about whether the file aliased the import.
    """
    def __init__(self, tree: ast.AST) -> None:
        self._bindings: dict[str, str] = {}
        for node in ast.walk(tree):
            if isinstance(node, ast.Import):
                for alias in node.names:
                    name = alias.asname or alias.name.split(".")[0]
                    self._bindings[name] = alias.name
            elif isinstance(node, ast.ImportFrom):
                if not node.module:
                    continue
                for alias in node.names:
                    name = alias.asname or alias.name
                    self._bindings[name] = f"{node.module}.{alias.name}"

    def module_for(self, bound_name: str) -> str | None:
        return self._bindings.get(bound_name)

    def canonical_for(self, expr: ast.expr) -> str | None:
        """For Name or Attribute chains, return the canonical dotted path."""
        parts: list[str] = []
        cur: ast.expr | None = expr
        while isinstance(cur, ast.Attribute):
            parts.append(cur.attr)
            cur = cur.value
        if not isinstance(cur, ast.Name):
            return None
        head = self._bindings.get(cur.id)
        if head is None:
            return None
        if not parts:
            return head
        parts.reverse()
        return f"{head}.{'.'.join(parts)}"
```

```python
def is_logger_call(call: ast.Call, attrs: set[str]) -> bool:
    """True if `call` is `<receiver>.<method>(...)` where method is in `attrs`."""
    func = call.func
    if not isinstance(func, ast.Attribute):
        return False
    return func.attr in attrs
```

### Story S5 — security_policy_a AST scan

**Audit IDs closed:** S-A4 (eval/exec aliased), S-A5 (multi-line
shell=True), S-A8 (trailing-comment leakage).

**Files:** `.harness/checks/security_policy_a.py`.

**Approach:** Replace the line-by-line regex sweep with an AST walk:

```python
DANGEROUS_NAMES = {"eval", "exec", "__import__"}
DANGEROUS_DOTTED = {"os.system", "pickle.loads", "subprocess.getoutput"}
SUBPROCESS_DOTTED = {"subprocess.run", "subprocess.call",
                     "subprocess.Popen", "subprocess.check_call",
                     "subprocess.check_output"}

def _scan_dangerous_ast(path, source, tracker):
    tree = ast.parse(source)
    for node in ast.walk(tree):
        if isinstance(node, ast.Call):
            canonical = tracker.canonical_for(node.func)
            # eval/exec/__import__
            if isinstance(node.func, ast.Name) and node.func.id in DANGEROUS_NAMES:
                emit_dangerous(...)
            # os.system / pickle.loads / etc (canonical-aware)
            if canonical in DANGEROUS_DOTTED:
                emit_dangerous(...)
            # subprocess with shell=True (any argument arity / line wrap)
            if canonical in SUBPROCESS_DOTTED:
                if any(kw.arg == "shell" and isinstance(kw.value, ast.Constant)
                       and kw.value.value is True for kw in node.keywords):
                    emit_shell_true(...)
```

**Acceptance criteria:**

- `subprocess.run(\n    cmd,\n    shell=True,\n)` (multi-line) fires
  `Q13.dangerous-pattern`.
- `import os; os.system("foo")` fires (already covered).
- `from os import system; system("foo")` ALSO fires (new; closes
  S-AS-style aliasing for security_policy_a).
- A trailing comment containing literal `shell=True` does NOT fire
  (closes S-A8).
- Existing baseline entries remain valid; re-baseline only if AST
  scan finds a strict superset.

### Story S9 — CsrfProtect structural match + constructor middleware

**Audit IDs closed:** S-B3 (substring match), S-B4 (constructor-
pattern middleware missed).

**Files:** `.harness/checks/security_policy_b.py`.

**Steps:**

1. `_has_csrf_dep`: replace `"CsrfProtect" in ann_src` substring
   match with structural equality:

   ```python
   def _has_csrf_dep(fn) -> bool:
       targets = {"CsrfProtect"}  # consider making policy-driven
       for arg in list(fn.args.args) + list(fn.args.kwonlyargs):
           ann = arg.annotation
           if ann is None: continue
           name = _name_of(ann) if isinstance(ann, ast.Name) else None
           if name in targets: return True
           # Annotated[CsrfProtect, ...]
           if isinstance(ann, ast.Subscript):
               for slot in ast.walk(ann):
                   if isinstance(slot, ast.Name) and slot.id in targets:
                       return True
       return False
   ```

   Walk `Assign` and `Call` nodes whose result is bound to `app`. If a
   `Call` to `FastAPI(...)` has a `middleware=` keyword whose value
   is a list, and any element of that list contains a Name that ends
   with `CsrfMiddleware` or `CSRFMiddleware`, set `has_global_csrf =
   True`.

**Acceptance criteria:**

- `arg: NoCsrfProtectNeeded` no longer satisfies CSRF (closes S-B3).
- `app = FastAPI(middleware=[Middleware(CSRFMiddleware)])` exempts
  per-route CSRF (closes S-B4).
